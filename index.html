Instalacion

Copia este comando y pegalo en la Terminal de tu Mac
Comando de Instalacion

Abri la Terminal (Aplicaciones > Utilidades > Terminal) y pega este comando:

curl -fsSL https://raw.githubusercontent.com/patriciovallerino/n8n-lemon/main/instalar.sh | bash

El script instala Node.js y n8n automaticamente. Tarda 2-3 minutos.
